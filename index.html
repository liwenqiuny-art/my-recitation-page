快速部署指南
  1. 在 GitHub 新建一个 public 仓库（如 chudang-study）
  2. 上传 index.html 和 knowledge_structured.json 到仓库根目录
  3. 进入仓库 Settings → Pages → Source 选 main branch / root，保存
  4. 几分钟后访问 https://yourname.github.io/chudang-study/
  5. 在此页面填写 Token 和仓库信息，即可从网页直接编辑并同步
